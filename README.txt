現状はcmd等から"java ○○"で実行しなければ動作の確認は行えない。

Noaction
・マウスの無動作時間を検知してタスクトレイから未操作を報告するもの。
　(現状ではコードの内容を変更しなければ無動作時間の設定を行えない)

　→今後の展望
　　タスクトレイメニューなどから検出時間の操作を可能にする。
　　常駐化してcmdで動作させ続けなくても監視が行えるように変更する。

FileSearch
・指定したパスの配下に任意の単語が含まれるファイルがないかを検索し、
　それを任意のファイルにコピーするもの

　→今後の展望
　　検索するパス、調べる単語、コピー先のパス等を入力するためのウィンドウを表示する。
